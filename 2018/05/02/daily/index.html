【5.2更新】坑货的喜感日常

发表于 2018-05-02 | 分类于 我们
  • 昨天去接大坑，很晚才到。今天舍友问起，就说到了坑货坑队友和坑自己的喜感日常，所以打算记录一下。
  • 记录之事只是冰山一角，日常小坑不胜其数。
  • 记录下来的目的就是以后想起来开心一下。
  • 立贴为证，实时更新。

  • 清明节假期，本来说要和闺蜜去扬州的，结果三个人明明住的离南站很近，还是没赶上车。最后临时改成了去杭州，在杭州又碰上下雨，湿光了鞋子。。。想想都可怜，哈哈哈哈

  • 闺蜜来南京乘飞机，住在禄口机场附近。俩人在夫子庙吃饭，最后快到点的时候，明明离三号线比较近，路痴把人家带到了一号线，害的人家跑得喘不过气，她自己是笑的喘不过气、、、、、、我在想如果我这么不靠谱，她会不会爆炸，哈哈

  • 和严霞打算去吃捞王。计划的好好的突然老板来了个工作，哈哈哈哈哈哈哈，然后气鼓鼓的去吃了一碗面

  • 三联坑：

    • 五一回家，都已经到了车站了，马上就要上车的时间了，发现自己买错到达站了，然后又急急忙忙的去退票，改签了个半小时后的

    • 五一回来，先是买了2:30回来的票，又觉得太早，退票又慢慢吞吞，不但差一分钟多收了10%的手续费，还没有15:30的票了。所以之后买了16:40的车，大约八点到…..结果，高速路上堵车，这个坑货十点才到，我在宿舍等了一个半小时，在高铁站又等了一个半小时。把自己累够呛，还和我说“我觉得你也挺惨的”……哈哈哈哈哈哈，我这是被这个人的脑回路逗到了。最后和路痴一样不知道自己在那，绕了一大圈才找到。最后，我说打车吧？她说“为什么？”，一下我就不知道回什么了，（回宿舍杨天浩也说为什么不打车，我说“我说了打车，她说为什么”。然后杨天浩也说她说这个“为什么”确实挺骚的。）然后提着一个很重的包坐地铁回了宿舍。

    • 故事还没有结束，第二天早上说咱九点一起去教研室，八点四十闹钟响了，她说推后半小时吧。好， 等九点十五再起来，说“起床失败，对不住+1”，我……突然有一种遇人不淑的感觉。哈哈。觉得很少有人能回一趟家来回都是坑，坑完自己坑队友的。在下叶良辰实在是佩服。
